# faster-than-caml
Based on the game Faster Than Light

## Installation

The main external dependency is [Lambda Term](https://github.com/diml/lambda-term). 
It can be installed through opam:
```
opam install lambda-term
```

## Running

The game is designed to be played in a terminal with *black* background with *white* text,
in a full-screen window. To run, execute:

```bash
make 
make play
```
